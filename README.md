##  CCS-315: Numerical Methods Lab | Computer Science | 5th sem | DM College of Science

A repo for the Computer Science Students of DM College of Science | 5th Sem | Computer Networks.

##  Questions

1. Find the roots of the equation by bisection method.
2. Find the roots of the equation by secant method.
3. Find the roots of the equation by Regula-Falsi method.
4. Find the roots of the equation by Newton's method.
5. Find the solution of a system of nonlinear equation using Newton's method.
6. Evaluate the approximate value of finite linear using Gaussian/Romberg integration.

##  Solution

Every question is solved and packed in its respective folders labelled Q1, Q2 .... Q6
Each file contains a readme file where the program is described in detail.

##  Technologies Used

- C
